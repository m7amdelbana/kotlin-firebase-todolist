# About The project

FirebaseKotlin is a simple android application using Kotlin programming language example on:
- Firebase Auth
- Login
- Registration
- Firebase Database
- RecyclerView

# Content Of This project

- Login Form
- Registration Form
- RecyclerView (ToDo List)
- Utilities
